ایجاد یک مخزن جدید در GitHub

ابتدا، باید یک مخزن جدید در GitHub ایجاد کنید. برای این کار، به سایت GitHub بروید و روی دکمه “New repository” کلیک کنید. سپس، نام مخزن را وارد کنید و آن را به صورت عمومی یا خصوصی تنظیم کنید. در نهایت، روی دکمه “Create repository” کلیک کنید.

مقدماتی در کامپیوتر شما

سپس، باید به دایرکتوری پروژه خود در کامپیوتر بروید. این کار را می‌توانید با استفاده از خط فرمان انجام دهید. برای مثال، اگر پروژه شما در دایرکتوری C:\Users\YourName\YourProject قرار دارد، می‌توانید دستور زیر را وارد کنید.

cd C:\Users\YourName\YourProject

ایجاد یک مخزن محلی Git

حالا که در دایرکتوری پروژه هستید، می‌توانید یک مخزن محلی Git ایجاد کنید. برای این کار، دستور زیر را وارد کنید. این دستور یک مخزن محلی Git در دایرکتوری فعلی شما ایجاد می‌کند.

git init

اضافه کردن فایل‌ها به مخزن Git

بعد از ایجاد مخزن، باید فایل‌ها و پوشه‌های پروژه خود را به آن اضافه کنید. برای این کار، دستور زیر را وارد کنید. این دستور تمام فایل‌ها و پوشه‌ها را به مخزن Git اضافه می‌کند.

git add .

ایجاد یک commit

حالا که فایل‌ها را اضافه کردید، باید یک نقطه عطف یا “commit” ایجاد کنید. برای این کار، دستور زیر را وارد کنید. این دستور یک نقطه عطف جدید با پیام “commit initial” ایجاد می‌کند.

git commit -m "commit initial"

اضافه کردن مخزن GitHub به عنوان یک “remote”

بعد از ایجاد نقطه عطف، باید مخزن GitHub خود را به عنوان یک “remote” به مخزن محلی خود اضافه کنید. برای این کار، دستور زیر را وارد کنید. در این دستور، Your-Repo-URL باید با URL مخزن GitHub خود جایگزین شود.

git remote add origin Your-Repo-URL

ارسال تغییرات به مخزن GitHub

در نهایت، تغییرات خود را به مخزن GitHub ارسال می‌کنید. برای این کار، دستور زیر را وارد کنید. این دستور، تغییرات شما را به شاخه “master” در مخزن GitHub ارسال می‌کند.

git push -u origin master
